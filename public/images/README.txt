ИНСТРУКЦИЯ ПО ЗАГРУЗКЕ ФОТОГРАФИЙ (ООО "ЕвроЗаборы")

Шаг 1: ПОДГОТОВКА ФОТО
- Сделайте фото ваших работ. Желательно использовать горизонтальные фото высокого качества.
- Переименуйте файлы на латинице (например: my_work_1.jpg, gate_red.png). 
- Избегайте кириллицы и пробелов в названиях файлов.

Шаг 2: ЗАГРУЗКА
- Переместите файлы в текущую папку (/public/images/).
- Если вы работаете через GitHub: Нажмите "Add file" -> "Upload files" и перетащите фото сюда.

Шаг 3: ПРИВЯЗКА К САЙТУ
- Откройте файл /data/catalog.ts.
- Найдите нужную работу в списке catalogData.
- В поле "url" впишите путь: "/images/имя_вашего_файла.jpg".
- Сохраните файл. Изменения появятся на сайте автоматически.

ПОДСКАЗКА:
Если вы хотите добавить работу, которой нет в списке, просто скопируйте блок { ... } целиком, 
поменяйте ID на следующий по порядку и впишите свои данные.
