Эта папка создана для ваших фотографий.

Как добавить фото на сайт:
1. Загрузите сюда ваши файлы (например, gate1.jpg, zabor2.jpg).
   На GitHub: Add file -> Upload files.
   
2. Откройте файл data/catalog.ts.

3. Измените ссылку на картинку.
   Было: image: "https://..."
   Стало: image: "/images/gate1.jpg"

Важно: имена файлов должны совпадать точь-в-точь (включая .jpg или .png).
